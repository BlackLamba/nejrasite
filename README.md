## 🛠️ Развертывание и Хостинг

Сайт разработан как ультра-быстрое, автономное одностраничное веб-приложение (SPA) с CSS-стилями внутри файла. Это гарантирует:
1.  **100% работоспособность при оффлайн-превью** в любых песочницах и файловых менеджерах (без загрузки внешних стилей/шрифтов с заблокированных CDN).
2.  **Простой хостинг:** Достаточно просто скопировать файл `index.html` на любой веб-сервер (Nginx, Apache, GitHub Pages, Yandex Cloud Object Storage).

Для запуска на сервере под управлением Nginx используйте стандартный конфигурационный файл:

```nginx
server {
    listen 80;
    server_name neira-tech.ru www.neira-tech.ru;

    location / {
        root /var/www/neira;
        index index.html;
        try_files $uri $uri/ =404;
    }
}
```
